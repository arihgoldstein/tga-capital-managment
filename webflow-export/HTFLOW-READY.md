# HTFlow Import Instructions

## How to Use HTFlow in Webflow

1. Open your site in **Webflow Designer**
2. Go to **Apps** panel (left sidebar) → Search **"HTFlow"**
3. Install and open HTFlow
4. You'll see two input fields: **HTML** and **CSS**
5. Copy the content from the files below and paste them
